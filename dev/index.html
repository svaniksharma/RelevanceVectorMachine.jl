Functions

RelevanceVectorMachine.RVM — Type
RVM

An instance of a relevance vector machine. Can be instantiated using rvm.

source
RelevanceVectorMachine.rvm — Function
rvm(formula::FormulaTerm, data, mode = "regression", max_iters = 1000)

Initialize and train a relevance vector machine, using the variables specified in formula with the data provided in data. mode can either be "regression" or "classification".

source
RelevanceVectorMachine.predict — Method
predict(rvm::RVM, X)

Given a table X and relevance vector machine rvm, compute predictions for X.

source
RelevanceVectorMachine.posterior — Method
posterior(rvm::RVM)

Returns a normal distribution with mean μ and covariance Σ corresponding to the distribution of the weight parameter (a vector) of the relevance vector machine.
